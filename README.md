Finance-Tracker
A Python + PySide6 desktop application that helps students track expenses, visualize spending, and automatically categorize expenses using AI.

## Features
- Add income and expenses
- AI-based expense categorization
- Expense analysis using graphs
- Budget tracking and overspending alerts
- Transaction history
- CSV-based data storage

  ## Tech Stack
- Python
- PySide6
- scikit-learn
- matplotlib
- CSV

## Future Scope
- Cloud backup and sync
- User login system
- Mobile version
